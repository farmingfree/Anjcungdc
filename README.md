# Anjcungdc

\# Ăn Gì Cũng Được



AI-powered food discovery mobile app helping users decide what to eat nearby using location, maps data, and smart recommendations.



\---



\## 🚀 Vision



Giúp người dùng không còn mất thời gian suy nghĩ “ăn gì hôm nay”.



\---



\## 🛠 Tech Stack



\- Flutter (Mobile App)

\- FastAPI (Backend API)

\- PostgreSQL + PostGIS (Location Database)

\- Python Crawlers (Data Pipeline)

\- AI Recommendation Engine



\---



\## 📁 Project Structure



\- mobile/     Flutter mobile app

\- backend/    Backend services

\- crawler/    Data crawling pipeline

\- docs/       Product \& technical docs
